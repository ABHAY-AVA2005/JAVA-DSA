# [Pair cube count](https://www.geeksforgeeks.org/problems/pair-cube-count4132/1)
## Easy
Given a positive integer n, count all pairs of ‘a’(&gt;=1) and ‘b’(&gt;=0) that satisfy the condition a3 + b3 = n.
Example :
Input: n = 9 
Output: 2
Explanation: There are two solutions: (a=1, b=2) and (a=2, b=1).
Input: n = 27
Output: 1
Explanation: Thereis only one solution: (a=3, b=0). 
Constraints:1 &lt;= n &lt;= 105
